# 对接AI Vtuber

ws端口默认为22336  

加载请使用 live2d数字人,大语言模型聊天(edge_tts，无须第三方tts) live2d_llm_edge_tts.html 这个页面  

第一次运行完成后，先使用 测试音频 播放一下先，在进行对接使用。  

AI Vtuber需要把out路径URL映射出来，才能通过url获取ai vtb合成的音频。
